流程控制

go语言中常用的流程控制就是if和for，此外还提供了switch和foto，但是只是基于for和switch的扩展流程控制

if-else条件控制

写法为：

if 表达式 {} else {}

不像c++，这里的{}是不能省略的

有一种特殊的写法是if a:=19; a>8 {}这种写法的好处是a是一个在if块中的局部变量

for循环控制

go语言中只有一种循环控制语句，就是for，其基本格式为：

for 初始语句; 条件表达式; 结束语句 {}

这三个语句都是可以省略的(但;是有时可省，有时不可省)。所以虽然go没有while，但是可以模拟出来

for 条件表达式 {}
就相当于普通的while语句

for {}
就相当于无限循环

for range(键值循环)

go语言中可以用for range来实现c++中foreach的效果，去遍历数组，切片，字符串，map，通道等

基本写法为eg.:

for s, err := range str {}

其返回值有如下的规律：
1. 数组，切片，字符串会返回索引和值。
2. map会返回键值和值。
3. 通道只返回通道内的值。

switch条件控制

基本写法为：

switch 条件 {
case 值1:
case 值2:
default:
}

有个很不常见的用法是在case里面可以写fallthrough，可以执行下面一个就算条件不满足的case

golang的break语句

Go语言的break语句除了cpp那种跳出当前循环的用法外，还可以指定一个标签来决定到底要跳出哪一层循环

就比如说： 

package main

import (
        "fmt"
)

func main() {
loop:
        for {
                for i := 0; i < 10; i++ {
                        fmt.Println(i)
                        if i > 5 {
                                break loop
                        }
                }
        }
        fmt.Println("跳出了外层循环")
}
输出结果为:
0
1
2
3
4
5
6
跳出了外层循环

如果是普通的break不指定标签的话，将会无限循环

goto(跳转到指定标签)

这个也很少用，就是在程序的某个位置写一个标签，格式是：
标签1:
在程序中就可以：
goto 标签 1
进行跳转。
