变量的声明及赋值

go语言变量的声明方式及赋值方式有以下几种

  1. 标准声明

    var 变量名 变量类型=变量值

    例如，var a int = 5

  2. 类型推导赋值

    var 变量名=变量值

    例如 var a=5.0

  3. 短变量声明

    变量名:=变量值

    例如 a:=5

    请注意只要见到:=就一定是一个声明语句，而不是一个赋值语句

匿名变量用_来表示，就比如说函数返回了两个参数(go的函数是可以返回多个参数的)，如果你不想要某一个参数，就可以用_来占位

常量

声明的格式是const 常量名=常量，只能用标准声明或者类型推导声明，不能用短变量声明

可以批量声明const变量

比如说: 

const(
	a=5
	b=4
)

批量声明的时候，值可以省略，会默认与上面一行的常量值相等

常量计数器iota

iota是go语言的常量计数器，只能在常量的表达式中使用

iota在const关键字出现时重置为0,const中每新增一行常量声明将使iota 加一。可以理解为const语句中的行索引,注意是行的计数器

比如说： 

const(
        a=iota //a=0
        b=iota //b=1
        c=iota //c=3
)
