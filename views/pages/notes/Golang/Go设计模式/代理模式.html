代理模式(Proxy)

代理模式，为其他对象提供一种代理以控制对这个对象的访问

代理模式就是你让别人去帮你做一件事情。就比如说我是一个特工，我不想暴露自己的声音，我就让别人帮我讲话。当然这个别人在讲话的时候还可以对我的语言进行一定的加工

代理模式的示意图如下：

代理模式的代码： 

package Proxy

import "fmt"

type Subject interface {
        Request()
}

type realClient struct{}

func (r *realClient) Request() {
        fmt.Println("我是小胖")
}

type Proxy struct {
        realclient *realClient
}

func NewProxy() *Proxy {
        return &Proxy{}
}

func (p *Proxy) Request() {
        p.realclient.Request()
}

package main

import (
        "../Proxy"
)

func main() {
        p := Proxy.NewProxy()
        p.Request()
}

输出结果为：
我是小胖

可以看到在main包里面，我并不知道特工真正的身份。但是我通过特工的代理，知道了特工的名字是小胖

实现的关键：一个是特工和代理都要实现同一个接口(为了传参数，虽然上面的例子没有)。一个是代理类要包含特工类
