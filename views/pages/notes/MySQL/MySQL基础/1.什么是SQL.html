SQL简介

数据库和数据库软件

数据库和数据库软件：数据库是通过数据库软件创建和操纵的容器

表(table)

表：某种特定类型数据的结构化清单

表就像文件夹一样用来存储某种特定类型的数据。表具有一些特性，这些特性定义了数据在表中如何存储，比如存储什么样的数据，数据如何分解，各部分信息如何命名等，这些特性被称为模式(schema)

列(column)

列：表中的一个字段。所有表都是由一个或多个列组成的

表由列组成。列中储存这表中某部分的信息。可以想象表是一个表格，列就是表格的一列，比如姓名、年龄列

数据库的每个列都应该有相应的数据类型(datatype)比如数字，文本，时间等

行(row)

行：表中的一个记录

就类似于表格的一行，会代表一条记录，所以行也可以称为记录(record)

主键(primary key)

主键：一列(或多列)，其值能够唯一区分表中的每一行

主键其实就是某一个特定的列。但这列的数据是每一行的唯一的，比如学号

一般来说有以下几点习惯：

  • 不更新主键列中的值
  • 不重用主键列的值
  • 不在主键列中使用可能会更改的值

SQL(Structured Query Language)

SQL是结构化查询语言。SQL是一种专门用来与数据库通信的语言
